== Description ==
The Only Schema Product Reviews plugin enables you to add review schema to posts. The plugin doesn't modify the design of the site and adds schema only to the posts of your choice. The generated schema will be visible only to search bots.

<h3> How to use the plugin? </h3>
Once you install the Only Schema Product Reviews plugin and activate it, open a post of your choice. You'll find a box with drop a down list below the post editor. Select "Yes" option if the post is a review else ignore the drop-down list if the post isn't a review. Once you select "Yes", you'll see several boxes below the dropdown list. Enter the details and save/publish/update the post.
